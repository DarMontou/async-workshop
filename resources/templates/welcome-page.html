Welcome

Welcome to Get going with core.async. The goal of this workshop is to help you become familiar with core.async to the level where you can start using it in your own projects.

In particular, this worksop should provide:

  1. An overview of the core.async library primitives
  2. A catologue of usage patterns for building applications with core.async
  3. Practical experience of building a simple chat application using core.async

Requirements

In order to complete this course, you will need this following:

  • Familiarity with the Clojure programming language
  • A working Clojure or ClojureScript environment
  • The code repository clone from FIXME

Getting started

Once you have obtained the source code for the workshop, there are a number of places you can go:

  1. You can jump right into the tutorial, which will take you step by step in building either a chat client in ClojureScript or a chat server in Clojure using core.async
  2. You can browse the included reference, to become more familiar with core.async and the patterns.
  3. Finally, you can check out the chat demo, giving you a glimpse of what you will build in the tutorial.
